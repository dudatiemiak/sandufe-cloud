## Como executar

1. Clone o repositório:

```bash
git clone https://github.com/dudatiemiak/sandufe-cloud.git
```

2. Entre na pasta do projeto:

```bash
cd sandufe-cloud
```

3. Dê permissão e execute o script:

```bash
chmod +x ./script.sh
./script.sh
```
